# App Information

## Nama Aplikasi
Simple Calculator

## Deskripsi Pendek
Aplikasi kalkulator sederhana berbasis Android untuk perhitungan matematika dasar.

## Deskripsi Panjang
Simple Calculator adalah aplikasi Android ringan yang dirancang untuk membantu
pengguna melakukan perhitungan matematika dasar seperti penjumlahan, pengurangan,
perkalian, dan pembagian. Aplikasi ini dibuat sebagai bagian dari proyek pembelajaran
pemrograman bergerak.
